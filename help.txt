Supported:
ComodoUpdate - WinVista+
ComodoUpdateXP - WinXP

Update Comodo Antivirus database and fix the date of the last update later, to not signal an error about outdated database.
Run manually or select an application in the Tasks Scheduler.
(taskschd.msc -> Tasks Scheduler Library -> COMODO -> Comodo Update -> Properties -> Actions -> Edit -> Browse Program, arguments must be empty)

Warning:
Comodo path hardcoded on "C:\Program Files\COMODO\COMODO Internet Security".
